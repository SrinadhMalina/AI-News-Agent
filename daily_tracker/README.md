# 🚀 Daily Hustle Tracker

A high-performance daily habit tracking application designed to drive growth and maintain discipline.

## Features
- **Daily Checklist:** Track essential daily tasks including health (supplements, water, gym), learning (German, AI), and mindfulness (meditation).
- **Growth Engine:** A "Finalize Day" reporting system that evaluates your performance and provides a direct verdict on your hustle.
- **Persistence:** Automatically saves progress to a local JSON database (`daily_data.json`).
- **Modern UI:** Built with `customtkinter` for a sleek, dark-mode interface.
- **Auto-Maximized:** Launches in zoomed mode for a full-screen productivity experience on Windows.

## Tech Stack
- **Language:** Python 3.11+
- **UI Framework:** `customtkinter`
- **Data Storage:** JSON

## How to Run
1. Ensure `customtkinter` is installed: `pip install customtkinter`
2. Run the tracker: `python daily_tracker/daily.py`

## The Verdict
- **🌟 GOOD FULL-FLEDGED DAY!** — Awarded only when 100% of tasks are completed.
- **❌ THIS IS NOT ENOUGH!** — Triggered if any task is left unfinished. Hustle more to dominate your goals.
